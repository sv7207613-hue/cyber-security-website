Cyber Security & Digital Privacy

Cyber Security refers to the practice of protecting systems, networks, and data from digital attacks. These attacks are usually aimed at accessing, changing, or destroying sensitive information.

Digital Privacy focuses on protecting personal data shared online. It ensures that user information is not misused, tracked, or leaked without permission.

Common Cyber Threats

Threat Description Impact
Phishing Fake emails or websites to steal data Data Theft
Malware Harmful software infecting systems System Damage
Ransomware Locks data and demands payment Financial Loss
Data Breach Unauthorized access to data Privacy Violation
